# Supply Chain RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using Flowise, Google Gemini, and Pinecone Vector Database to answer questions from supply chain and supplier performance documents.

## Project Overview

This chatbot allows users to interact with supply chain data using natural language. It retrieves relevant information from uploaded documents and generates accurate responses using Google's Gemini model.

## Features

- Retrieval-Augmented Generation (RAG)
- Document-based Question Answering
- Google Gemini Integration
- Pinecone Vector Database
- Flowise No-Code Workflow
- Semantic Search
- Conversational Chat Interface

## Technology Stack

- Flowise
- Google Gemini API
- Pinecone Vector Database
- Retrieval QA Chain
- Vector Embeddings
- RAG Architecture

## Setup Instructions

1. Configure Google Gemini API Key
2. Configure Pinecone API Key
3. Create a Pinecone index (Dimension: 3072)
4. Upload documents to Flowise Document Store
5. Connect Retriever and Gemini nodes
6. Test the chatbot

## Example Questions

- Who is the best-performing supplier?
- Which suppliers have a high risk level?
- Show all Tier-1 suppliers.
- Which supplier has the highest compliance score?
- Compare supplier performance across regions.
